## 从前端调用Rust
`Tauri`提供了一个简单而强大的 `command` 系统，用于`从 web 应用程序调用 Rust 函数`。 命令可以`接受参数并返回值`。 它们也可以返回错误并且是 `async`。

官方文档：https://tauri.app/v1/guides/features/command

## Tauri基础命令
启动
```shell
npm run tauri dev
```
编译
```shell
npm run tauri build
```
