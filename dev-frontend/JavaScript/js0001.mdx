## 实现禁用复制
```js
document.addEventListener('copy', e=>{
    e.preventDefault(); // 阻止copy事件默认行为
    // cliboardData剪切板对象
    e.clipboardData.setData('text/plain', '禁止复制'); // 替换复制文本自定义
})
```
## 实现解禁
移除这些事件就行了
<Tip>f12--事件监听器移除即可</Tip>
